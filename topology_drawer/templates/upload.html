How To:

  • Upload .json file from Tools PM or Tools JSON
  • After click "Submit" button, the file "network_custom.drawio" will downloaded
  • Then, open "https://app.diagrams.net/" and import the .drawio file
  • After that, select all object on draw.io, Select Arrange -> Layouts -> Vertical Tree
  • If the "Tree" didn't correct, you can use Tools JSON to reorder the json structure, and then repeat from 1st step
